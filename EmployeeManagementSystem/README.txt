User Guide: Employee Management System

Thank you for choosing the Employee Management System. This guide will provide you with step-by-step instructions on how to set up and use the program on your computer. Just so you know, the system uses MySQL as the database. Follow the directions below to ensure a smooth installation and usage process.

------------------------------------------------------------------------
------------------------------------------------------------------------

Open src/employeemanagementsystem folder to see the class-wise code.

------------------------------------------------------------------------
------------------------------------------------------------------------

System Requirements:
Operating System: Windows, macOS, or Linux
Java Development Kit (JDK) installed on your computer
MySQL database installed and running
MySQL Connector/J library added to your Java classpath

------------------------------------------------------------------------
------------------------------------------------------------------------

Installation Steps:
Download the Employee Management System files from the provided source.
You can extend the downloaded files to a preferred location on your computer.

------------------------------------------------------------------------
------------------------------------------------------------------------

Database Setup:
Open the MySQL database management tool (MySQL Workbench).
Create a new database for the Employee Management System.
Execute the provided SQL script to create the necessary tables and database schema.

------------------------------------------------------------------------
------------------------------------------------------------------------

necessary tables
*employeemanagmentsystem(database name at ConnectingDatabase class )

*employeedata( to store all the data at EmployeeAddPanel class)

*adminlogin(previously stored admin and password at LoginPage class)
 
*AdminLoginInfo( to store the admin login time date and admin name at AdminLoginDetails class )

*EmployeeLoginInfo ( to store the Employee login time date and Employee ID at EmployeeLoginDetails class )

*Applications( to store the employee application at ApplicationForm class )

------------------------------------------------------------------------
------------------------------------------------------------------------

Using the Employee Management System:

*On the welcome page, select whether you want to log in as an employee or an administrator.

*If logging in as an administrator, use the provided admin credentials to access the admin panel.

*In the admin panel, you can add, delete, and edit employee details, view login history, and manage the system.

*If logging in as an employee, you can view your profile and send applications to the administrator.

*Explore the different panels and functionalities of the Employee Management System using the intuitive GUI.

------------------------------------------------------------------------
------------------------------------------------------------------------

Note: Ensure that the MySQL database is running before launching the Employee Management System.
